def parse_task_from_text(text):
 """
 Parse a free-form task description into a structured Task object.

 This function extracts task metadata such as title, priority, tags,
 and due date from a loosely formatted string using inline markers.

 Supported markers:
 - "@tag" for tags
 - "!value" for priority (numeric or named)
 - "#value" for due dates

 Parameters:
  text (str): Input string containing task details.

 Returns:
  Task: Object containing:
   - title (str)
   - priority (TaskPriority)
   - due_date (datetime or None)
   - tags (list)

 Raises:
  NameError, AttributeError if dependencies are missing.

 Example:
  parse_task_from_text("Buy milk @shopping !2 #tomorrow")

 Notes:
  - First priority is used
  - First valid date is used
  - Invalid dates are ignored
  - Tags are not deduplicated
 """

4. Final Combined Documentation Version

def parse_task_from_text(text):
 """
 Parse a free-form task string into a structured Task object.

 This function reads inline markers from a task string and extracts
 structured data including title, priority, tags, and due date.

 Supported markers:
  @tag        → Tag
  !1–!4       → Priority levels
  !low/high   → Named priority
  #date       → Due date

 Behavior:
  - Remaining text becomes title
  - First priority is used
  - First valid date is used
  - Multiple tags supported

 Parameters:
  text (str): Task input string.

 Returns:
  Task: Object with extracted properties.

 Example:
  task = parse_task_from_text("Finish report !urgent @work #friday")

 Notes:
  - Markers require whitespace prefix
  - Tags are alphanumeric only
  - Invalid dates ignored
  - Title may be empty

 Limitations:
  - No complex tag formats
  - No validation of conflicting inputs
  - Duplicate tags not removed
 """
